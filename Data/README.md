# Data

This directory contains datasets generated and validated by the DART pipeline.

Each technique folder in Samples corresponds to one or more MITRE ATT&CK techniques and includes:
- **Raw outputs** from LLM generation (Initial)
- **Syntax and Semantic Validation** from Multi-Agent Jury Evaluation Results (LLM Jury)
- **Validated command samples** after syntax and semantic filtering (Syntax Checks)
- **Metadata files** with evaluation scores and diversity statistics
